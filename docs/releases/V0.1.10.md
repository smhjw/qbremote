# TorrentRemote 0.1.10

## 中文

- 首页新增实时上下行速度曲线，整体状态一眼可见。
- 优化服务器切换与后台恢复，减少旧数据残影和空白闪屏。
- Tracker 页面支持复制、编辑、删除，并可隐藏或显示 passkey。
- 改进种子详情、文件树浏览、返回定位和图标视觉细节。

## English

- Added a realtime upload/download speed chart to the home dashboard.
- Improved server switching and background restore to reduce stale data residue and blank flashes.
- Tracker pages now support copy, edit, delete, and passkey show/hide.
- Refined torrent detail navigation, file tree browsing, return positioning, and launcher visuals.
