This app requires login to the user's OWN qBittorrent server, not a developer-provided service. Users enter their server's host, port, username, and password in the app's connection settings. The app sends credentials directly to the user's own qBittorrent WebUI via its Web API. No developer-owned server is involved; all data flows only between the user's device and their own server.
